# **App Name**: MediaVault

## Core Features:

- Media Note Creation: Allow users to create notes that can contain various media types, including images, voice recordings, text, and scribbles.
- Media Organization: Implement a system for organizing notes into categories or tags, allowing users to easily find specific notes later.
- Passkey Protected Notes: Enable users to lock individual notes with a passkey, ensuring that only authorized users can access sensitive information. Store unlocked notes in a temporary holding place.
- Note Locking: Allow users to lock some notes with a password or biometric authentication to protect sensitive data.
- Note Storage: Cloud database for storing the notes with media.

## Style Guidelines:

- Primary color: A calm blue (#64B5F6) to provide a sense of security and focus.
- Background color: Light gray (#EEEEEE) for a clean and distraction-free note-taking environment.
- Accent color: A subtle orange (#FFAB40) to highlight important UI elements such as the lock/unlock buttons.
- Body and headline font: 'Inter' for a modern, readable, and neutral look suitable for long text.
- Use simple, clear icons to represent different media types and actions, such as a camera icon for images, a microphone icon for voice recordings, and a pen icon for scribbles.
- Maintain a clean and intuitive layout with a clear hierarchy of information, making it easy for users to create, organize, and access their notes.
- Implement subtle animations when locking and unlocking notes to provide visual feedback to the user.
